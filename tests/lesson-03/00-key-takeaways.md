## Javascript loop
1. Cú pháp vòng lặp for (i)
- for (<điều kiện khởi tạo>; <điều kiện lặp>; <cập nhật>) {
// code
}
        - Điều kiện khởi tạo: chạy một lần duy nhất, khi vòng lặp bắt đầu.
        - Điều kiện lặp: nếu đúng thì chạy tiếp, sai thì dừng.
        - Cập nhật: chạy vào mỗi cuối vòng lặp, để thay đổi giá trị của biến đếm.

## Javascript console log
1. Sử dụng với nháy đơn, nháy kép
        - console.log(‘Toi la Linh’);
        - console.log(“Toi la Phong”);

2. Sử dụng kèm với variable
        - let name = “Linh”;
        - console.log(`Toi la ${name}`);

3. Sử dụng cộng chuỗi
        - console.log(“Toi ten la” + name + “”)
